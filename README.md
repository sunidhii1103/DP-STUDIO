# ✨ Overview

DP Studio is a premium interactive platform designed to help students, interview candidates, and developers truly understand **Dynamic Programming** through synchronized visualizations, algorithm execution playback, state transitions, recursion trees, reconstruction paths, and side-by-side comparisons.

Instead of memorizing formulas, users *see* how DP evolves internally.

# 📚 Supported Algorithms

## ✅ Implemented

| Algorithm | Features |
|---|---|
| Fibonacci | Recursion → Memoization → Tabulation |
| Longest Increasing Subsequence | DP + Binary Search Optimization |
| Longest Common Subsequence | Table reconstruction |
| Matrix Chain Multiplication | Partition DP visualization |
| Edit Distance | Transformation tracking |
| 0/1 Knapsack | State transition comparison |
